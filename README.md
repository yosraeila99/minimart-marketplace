# MiniMart Marketplace

MiniMart Marketplace is a modern React-based online marketplace application that allows users to browse products, add items to cart, place orders, and manage their account activity. It also includes an admin dashboard for managing products and tracking orders.

## Project Overview

This project was built as a frontend portfolio project to demonstrate practical React skills, component structure, routing, state management, localStorage usage, and marketplace workflow logic.

The application simulates a small online food and products marketplace where customers can shop from different stores and admins can manage products and orders.

## Features

### User Features

- Browse products
- View product details
- Add products to cart
- Increase and decrease product quantity
- Remove products from cart
- Checkout form
- Place orders
- View order success page
- View user dashboard
- View order details
- Register account
- Login and logout

### Admin Features

- Admin login
- Protected admin dashboard
- Add new products
- Edit products
- Delete products
- Search products
- View recent orders
- Change order status
- View order details

### General Features

- React Router navigation
- 404 Not Found page
- Data persistence using localStorage
- Responsive layout
- Clean and modern UI
